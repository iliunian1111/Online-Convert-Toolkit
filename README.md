# Online-Convert-Toolkit
免费无广告、无需登录的在线格式转换工具，基于Python+Streamlit开发，轻量快速，电脑/手机端自适应，10秒搞定多格式互转，核心功能无使用次数限制！

> 🔗 **一键在线使用**：https://online-convert-toolkit.streamlit.app/
> 📱 **定制/批量功能**：微信添加 **hxxiln** 详询

## 🚀 核心功能
| 功能模块 | 支持能力 |
|----------|----------|
| 📄 PDF处理 | PDF转纯文本，快速提取内容 |
| 🔍 图片OCR | 识别图片中中英文文字，精准提取 |
| 🖼️ 图片格式互转 | JPG/PNG/WebP格式自由切换 |
| 🎬 音视频处理 | 视频提取音频（MP4转MP3） |
| 📊 办公文件转换 | Excel(xlsx/xls)转CSV格式 |
| 🧩 二维码工具 | 二维码生成、二维码内容解析 |
| 🔐 编码解码 | Base64文本编码/解码 |
| 🗜️ 小文件压缩 | 通用文件压缩为ZIP格式 |

## 📦 快速使用
### 在线使用（推荐）
直接点击链接打开即可使用：https://online-convert-toolkit.streamlit.app/
> 操作步骤：打开链接 → 选择对应功能 → 上传文件 → 等待处理 → 下载结果

### 本地部署运行
若需本地搭建，按以下步骤操作：
1. 克隆本仓库到本地
```bash
git clone https://github.com/你的用户名/Online-Convert-Toolkit.git
cd Online-Convert-Toolkit
```
2. 安装所需依赖
```bash
pip install -r requirements.txt
```
3. 启动本地服务
```bash
streamlit run app.py
```
4. 打开浏览器访问 `http://localhost:8501` 即可使用
